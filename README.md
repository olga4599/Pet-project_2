# Pet-project_2

Откройте сайт «‎Дом питомца» (http://158.160.56.133/app/pets) и на основе имеющихся в нем данных создайте конструктор класса Cat со следующими параметрами: имя, пол, возраст.

В отдельный файл импортируйте и создайте объект Cat, который выводит имеющихся на сайте котов, с одинаковыми параметрами, но с разными значениями.
